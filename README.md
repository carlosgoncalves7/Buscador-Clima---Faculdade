# Search for a city (Buscar cidade)

## Objetivo:
O app foi desenvolvido para buscar cidades do Brasil e consultar seu clima atual, como também a previsão de clima semanal

## Techs
HTML, CSS, JavaScript e Php

### HTML
Utilizado para desenvolver a estrutura da página

### CSS
Para aplicar estilos a página

### JavaScript
O javaScript para aplicar a interação entre o usuário

### PHP
Usado como um mini servidor para armazenar o histórico de cidades buscado na página
